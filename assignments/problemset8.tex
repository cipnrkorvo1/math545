	\section{}
	We begin with the assertion that $\mathbb{M}_n(\reals)$ is a vector space, defined previously with matrix addition and scalar multiplication.\\
	First we will show that $T(A) = A^T$ is a linear transformation.\\
	Let $A, B \in \mathbb{M}_n(\reals)$.
	\[ T(A + B) = (A + B)^T = A^T + B^T = T(A) + T(B) \]
	which satisfies additivity.\\
	Let $A \in \mathbb{M}_n(\reals)$ and $c \in \reals$.
	\[ T(cA) = (cA)^T = c \cdot A^T = c \cdot T(A) \]
	which satisfies scalar multiplication, proving $T$ is a linear transformation.\\
	Then, to further show that $T$ is an isomorphism of vector spaces, we must show that $T$ is bijective.\\
	\textbf{One-to-One:} Let $A, B \in \mathbb{M}_n(\reals)$ where $T(A) = T(B)$. Then $A^T = B^T$. If we take the transpose of both sides, we get $A = B$. We have shown $T(A) = T(B) \implies A = B$, therefore $T$ is one-to-one/injective.\\
	\textbf{Onto:} Let $A, C \in \mathbb{M}_n(\reals)$ and choose $A$ such that $A = C^T$. Then $T(A) = A^T = (C^T)^T = C$. We have shown $\forall C \in \mathbb{M}_n(\reals), \exists A \in \mathbb{M}_n(\reals)$ such that $T(A) = C$, which makes $T$ onto/surjective.\\
	Since we have shown that $T$ is injective and surjective, it is bijective, which completes the proof that it is an isomorphism of vector spaces.

    Recall that matrices $A$ and $B$ are similar if there exists an invertible matrix $P$ such that
    \[ B = P^{-1}AP \]
    We begin by taking the transpose of each side
    \[ B^T = (P^{-1}AP)^T \]
    We can apply transposition properties to get
    \[ (P^{-1}AP)^T = P^TA^T(P^{-1})^T = P^TA^T(P^T)^{-1}\]
    Now we must rearrange the terms to match the required form to show similarity. Let $Q = (P^T)^{-1}$. Because $P$ is invertible, so is $P^T$, hence $Q = (P^T)^{-1}$ is also invertible. Then
    \[ P^TA^T(P^T)^{-1} = Q^{-1}A^TQ \]
    Thus we have shown
    \[ B^T = Q^{-1}A^TQ \]
    for some invertible matrix $Q$, which means $A^T$ and $B^T$ are similar matrices.

    The eigenvalues of $A$ are solutions to
    \[ \det(A-\lambda I) = 0 \]
    We know $\det(M) = \det(M^T)$ for any square matrix $M$, thus
    \[ \det(A-\lambda I) = \det((A - \lambda I)^T)\]
    Distribute and simplify the transpose:
    \[\det((A-\lambda I)^T) = \det(A^T - \lambda I^T) = \det(A^T - \lambda I) \]
    Thus we have shown
    \[ \det(A-\lambda I) = \det(A^T - \lambda I) \]
    which means $A$ and $A^T$ have the same eigenvalues.\\
    \\
    We can show that $A$ and $A^T$ do not necessarily have the same eigenvectors with the following example:
    \[ \text{Let } A = \bmat{1&1\\0&1}, \text{ thus } A^T = \bmat{1&0\\1&1}. \]
    The eigenvectors of $A$ can be found by solving $(A-I)\vec{v} = 0$.
    \[ \bmat{0&1\\0&0}\bmat{x\\y} = \bmat{0\\0} \longrightarrow \bmat{y\\0} = \bmat{0\\0} \implies y = 0, \text{ so } \vec{v} = \bmat{x\\0}\]
    The eigenvectors of $A^T$ can be found by solving $(A^T-I)\vec{v} = 0$.
    \[ \bmat{0&0\\1&0}\bmat{x\\y} = \bmat{0\\0} \longrightarrow \bmat{0\\x} = \bmat{0\\0} \implies x = 0, \text{ so } \vec{v} = \bmat{0\\y} \]
    Thus we have shown that even though $A$ and $A^T$ have the same eigenvalue, they may not have the same eigenvectors.
