	\section{}
	Recall that a matrix $A$ is diagonalizable if $D = P^{-1}AP$ where $D$ is a diagonal matrix.\\
	The inverse matrix is given by 
	\[ P^{-1} = \bmat{\frac{1}{5} & 0 & \frac{2}{5}\\0&-\frac{1}{4}&0\\-\frac{2}{5}&\frac{1}{4}&\frac{1}{5}} \]
	so that
	\[ D = P^{-1}AP = \bmat{ 3&0&0\\0&2&0\\0&0&-2 } \]
	$D$ is a diagonal matrix, therefore the matrix $A$ is diagonalizable.
